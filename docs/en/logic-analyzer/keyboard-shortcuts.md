# Keyboard shortcuts

<!-- Source: Acute_TLmanual_en.pdf, Chapter 2, Page 41 -->

Quick keyboard shortcuts for Logic Analyzer operations.

## Available shortcuts

| Function | Key |
|----------|-----|
| Move to cursor position | A-Z |
| Add a cursor to the mouse position | Shift + A-Z |
| Start capture | Enter |
| Stop capture | ESC |
| Search | F3 or Ctrl+F |
| Zoom in on waveform area | Number Pad + |
| Zoom out on waveform area | Number Pad - |

---

## Cursor shortcuts

### Navigation

Press any letter key (A-Z) to quickly jump to the corresponding cursor position in the waveform area.

### Adding cursors

Press Shift + any letter key (A-Z) to add a cursor at the current mouse position. If the cursor already exists, it moves to the mouse position without requiring a drag operation.

**Note:** The T cursor is reserved for the trigger point and cannot be repositioned using shortcuts.

---

## Capture control

### Start capture

Press **Enter** to begin data capture with the current settings.

### Stop capture

Press **ESC** to stop an ongoing capture operation.

---

## Waveform navigation

### Zoom operations

- **Zoom in:** Press **Number Pad +** to increase waveform magnification
- **Zoom out:** Press **Number Pad -** to decrease waveform magnification

Alternatively, use the mouse wheel for smooth zooming:

- Scroll up to zoom in
- Scroll down to zoom out

### Search function

Press **F3** or **Ctrl+F** to open the search dialog. Use search to find specific:

- Signal edges
- Value matches
- Pulse widths
- Time positions

Learn more: [Cursor operations](cursor.md#waveform-search)

---

## Tips

- Combine shortcuts with mouse operations for efficient navigation
- Use Shift + letter keys to quickly place multiple measurement cursors
- The Enter/ESC keys work from anywhere in the application window
