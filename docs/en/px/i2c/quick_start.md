# Quick Start
FOLLOWING STEPS:
1. Activate [Selecting Bus](bus_select.md).
2. Set up the [PX pod parameters](px_pair_settings.md).
3. *(Optional)*Add your [Internal node](add_internal_node.md) to topology.
4. *(Optional)*Set up your [DC output](../px.md#power-supply).
5. [Decode settings](controller.md#decode-settings) for I2C signals analysis.
6. [Assign your topology](controller.md#assign).
7. *(Optional)*[Scan all available address](controller.md#scan-external-node)(0x08 ~ 0x77) in [Controller mode](controller.md) only.
8. Send commands using [I2C wizard](controller.md#send-packet) in [Controller mode](controller.md) only.
