# Working with Multiple Sessions

Open additional session tabs to work with multiple analyzer modes simultaneously.

- **Add Logic Analyzer** ![](/assets/misc/add-new-la.png): Open a new [Logic Analyzer](../logic-analyzer/index.md) session
- **Add Protocol Analyzer** ![](/assets/misc/add-new-pa.png): Open a new [Protocol Analyzer](../protocol-analyzer/index.md) session

You can switch between tabs by clicking the tab without closing the current session. This also allows you to leave the device always connected between each session tasks.
