
# Demo mode

If the software shows "Demo mode" during execution, try the following steps to resolve the issue:

### Step 1: Install the latest software version

1. Visit the official Acute Technology Inc. website
2. Navigate to **Download** → **Software**
3. Select **[Logic Analyzer] Logic Analyzer (TL3000, TL4000 series)**
4. Download and install the latest version

### Step 2: Check USB cable

Use the original USB3.0 cable included in the kit. Third-party cables may cause connection issues.

### Step 3: Verify driver status

1. Connect the device USB cable to your computer
2. Open the Device Manager in your operating system
3. Check if **Acute USB BootLoader** or **Acute USB3.0 Product** appears

**If the driver is missing or shows an error:**

1. Visit the Acute website: **Download** → **Software**
2. Download the USB3.0 driver package
3. Follow the troubleshooting manual included in the package to reinstall the driver

### Step 4: Reset the connection

1. Remove all probes from the device
2. Unplug and re-plug the USB3.0 cable
3. Restart your computer
4. Check if the driver appears in Device Manager

### Step 5: Contact support

If the issue persists after completing the above steps, please contact us with:

- Your device model number
- Software version
- Operating system details
- Description of the issue
- Any error messages displayed
