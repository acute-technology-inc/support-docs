# Info Panel

Optional panel showing detailed information about channels and triggers.

**Location:** Right side of the window (can be toggled)

**Information displayed:**

1. **Channel Information**
   - Channel assignments
   - Current logic levels
   - Activity status

2. **Value Information**
   - Digital values (0/1)
   - Analog voltage (if applicable)
   - Bus values (hex/decimal/binary)

3. **Trigger Information**
   - Current trigger configuration
   - Trigger conditions per channel
   - Pass count settings

**Toggle visibility:**

Access through menu: View → Info Panel (or system environment settings)

**Use cases:**

- Quick reference during setup
- Monitoring live values
- Verifying trigger configuration
